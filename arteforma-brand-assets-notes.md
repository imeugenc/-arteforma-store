# ArteForma Brand Assets

## Included files

- `arteforma-af-monogram.svg`: standalone AF icon
- `arteforma-logo-lockup.svg`: icon plus wordmark and tagline
- `arteforma-tiktok-outro.svg`: vertical TikTok end screen

## Direction

The monogram is intentionally angular and bold so it survives glow, animation and small-size use.
It takes cues from luxury-tech branding and should work well for video outros, stickers, prints and web.
