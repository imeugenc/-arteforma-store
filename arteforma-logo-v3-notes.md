# ArteForma Logo v3

Direction:

- modern minimalist
- urban premium
- sharp geometric AF
- clear A and clear F
- black and gold

Files:

- `arteforma-logo-v3-monogram.svg`: centered vector monogram, transparent background
- `arteforma-logo-v3-lockup.svg`: full logo for light backgrounds
- `arteforma-logo-v3-on-black.svg`: full logo for dark backgrounds

Design choice:

The `A` is now a conventional geometric letterform so it reads instantly.
The `F` is integrated as a separate, sharp vertical block instead of being twisted into the `A`.
